# svgo-extra

A collection of SVGO Plug-ins.

# Features

- make SVG's color easy to change.
- make SVG's size responsive.

# Usage

```bash
# pnmp
$ pnpm add svgo-extra

# yarn
$ yarn add svgo-extra

# npm
$ npm i svgo-extra
```
